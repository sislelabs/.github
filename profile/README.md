# Sisle Labs

Tools for indie hackers and micro-SaaS builders who looked at the pricing page,
did the math, and quietly opened a terminal instead.

We ship the boring infrastructure so you can get back to the part where you
rewrite the landing page for the fourth time this week.

---

### What's in the lab

🛰️ **[kuso](https://github.com/sislelabs/kuso)** — a self-hosted, agent-native
PaaS on real Kubernetes. Like the big cloud, minus the bill that makes you feel
things. Apps sleep when idle, autoscale when busy, and every button in the UI is
also a CLI command *and* an MCP tool — so you can deploy by hand, by script, or
by politely asking a robot.

🏗️ **payload-scaffold** — a Payload 3 + Next.js starter that's already
SEO-complete, i18n-ready, and self-hostable, so your "weekend project" can reach
production before the weekend technically ends. Clone, rename, ship.

🤿 **scubatony** — a dive shop's booking + ops platform, and living proof the
scaffold survives contact with an actual business. One Postgres, no
microservices, zero regrets.
